## Test gesamter Programmablauf (Neues Spiel)

1. Programm DnD.jar starten // das Programm startet ordnungsgemäß als Fenster mit Musik
2. Klicken Sie auf "Neues Spiel" // Charaktererstellung öffnet sich
3. Erstellen Sie einen Charakter: Geben Sie einen Namen ein, 
   wählen Sie eine Rasse aus (Elf, Orc etc.), klicken Sie auf den Button "Charakter übernehmen
   // das Intro öffnet sich mit Lauftext
4. Klicken Sie auf Weiter // Weltkarte öffnet sich, Musik ändert sich
5. Gehen Sie mit der Maus oben links auf das Bild (Lederbeutel), es erscheint ein Hinweistext 
   (Tooltip) mit der Aufschrift "Inventar" -> klicken Sie darauf // das Inventar öffnet sich
6. Klicken Sie im Inventar unten auf den Knopf "Schließen" // Sie sind wieder auf der Weltkarte
7. Beenden Sie das Programm oben rechts mit dem X
